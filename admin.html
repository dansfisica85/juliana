Sobre o painel

Este site é estático (hospedado no GitHub Pages). Por isso, o login e os dados (fotos, galeria, inscrições) ficam armazenados apenas no navegador onde você acessar. Para uma autenticação real e armazenamento em servidor, é necessário um backend.
